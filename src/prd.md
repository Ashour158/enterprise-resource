### Color Strategy
**Color Scheme Type**: Triadic with carefully balanced business colors

**Primary Color**: Deep Blue (#2563EB) - Trust, intelligence, enterprise reliability
**Secondary Colors**: 
- Success Green (#059669) - Revenue growth, successful deals, positive metrics
- Warning Amber (#D97706) - Risk factors, attention items, pending actions
- Accent Purple (#7C3AED) - AI insights, predictive analytics, premium features

**Color Psychology**: Blue establishes trust and reliability essential for enterprise sales tools. Green celebrates success and growth metrics. Purple indicates advanced AI capabilities. Amber draws attention to important items requiring action.

**Color Accessibility**: All color combinations meet WCAG AA contrast requirements (4.5:1 minimum).

**Foreground/Background Pairings**:
- Background (White #FFFFFF) + Foreground (Dark Gray #1F2937) = 12.6:1 contrast
- Primary Blue (#2563EB) + White text = 4.8:1 contrast  
- Success Green (#059669) + White text = 4.9:1 contrast
- Card Background (Light Gray #F9FAFB) + Foreground (Dark Gray #1F2937) = 11.8:1 contrast

### Typography System
**Font Pairing Strategy**: Single font family (Inter) with comprehensive weight variations for consistent enterprise appearance.

**Typographic Hierarchy**: 
- Headlines: Inter Bold 24-32px for dashboard titles and main sections
- Subheadings: Inter Semibold 18-20px for card titles and subsections  
- Body Text: Inter Regular 14-16px for content and descriptions
- Captions: Inter Medium 12-13px for labels and metadata
- Code/Data: JetBrains Mono for IDs, API responses, and technical data

**Font Personality**: Inter provides excellent screen readability with professional character suitable for data-heavy business applications.

**Readability Focus**: 1.5x line height for body text, generous letter spacing for data tables, adequate padding around interactive elements.

**Typography Consistency**: Consistent sizing scale based on 4px baseline grid with mathematical relationships between text sizes.

**Which fonts**: Inter for all UI text, JetBrains Mono for technical data display

**Legibility Check**: All fonts tested at minimum sizes across devices with sufficient contrast ratios maintained.

### Visual Hierarchy & Layout
**Attention Direction**: AI insights and critical deal information receive visual priority through color, sizing, and positioning.

**White Space Philosophy**: Generous spacing prevents cognitive overload in data-rich interfaces while maintaining information density for efficiency.

**Grid System**: 12-column responsive grid with consistent 24px gutters and 16px baseline grid for precise alignment.

**Responsive Approach**: Mobile-first design with progressive enhancement for tablet and desktop deal management workflows.

**Content Density**: Information-rich layouts optimized for professional users who need comprehensive data access with efficient scanning patterns.

### Animations
**Purposeful Meaning**: Smooth transitions reinforce deal progression through pipeline stages, while micro-animations provide feedback for AI processing.

**Hierarchy of Movement**: Pipeline stage transitions receive primary animation focus, with secondary animations for data updates and user interactions.

**Contextual Appropriateness**: Subtle, professional animations that enhance rather than distract from serious business decision-making.
